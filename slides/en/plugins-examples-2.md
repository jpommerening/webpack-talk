# Plugins

## `ExtractTextPlugin`

- Extracts text/code (CSS, for example) into an external file during build

## `NgAnnotatePlugin`
<!--{data-bespoke-bullet}-->

- Writes `ng-annotate` annotations for safe minification

## `PrefetchPlugin`
<!--{data-bespoke-bullet}-->

- Can populate cache with modules before they are imported
